Personalização de Prêmios de Seguro com Base na Qualidade do Sono

Características do Projeto

Este projeto aborda o desafio de personalizar prêmios de seguro saúde com base na qualidade do sono dos clientes. A ideia é utilizar dados de saúde, incluindo métricas de sono, para criar um modelo preditivo que avalie o risco individual de cada cliente e ajuste o prêmio de acordo.

O projeto foi estruturado em três sprints:

  1. Preparação dos dados: coleta, limpeza e análise exploratória.
  2. Desenvolvimento do modelo: seleção e treinamento de algoritmos de classificação.
  3. Integração e testes: integração do modelo ao sistema da seguradora e validação.

Técnicas Utilizadas

  • Linguagem Python para análise e modelagem.
  • Bibliotecas Pandas e Scikit-learn para manipulação de dados e machine learning.
  • Modelos de Machine Learning: Random Forest, Decision Tree e Regressão Logística.
  • Métricas de avaliação: Acurácia, Precisão, Recall, F1-Score, AUC-ROC e Matriz de Confusão.
  • Validação cruzada para otimização de hiperparâmetros e evitar overfitting.

Resultados

  • O modelo de Regressão Logística apresentou a melhor performance, com 95% de acurácia.
  • Implantação de um modelo inovador para precificação de prêmios de seguro.
  • Possibilidade de oferecer prêmios personalizados aos clientes, melhorando sua experiência.
  • Contribuição para a conscientização sobre a importância da qualidade do sono na saúde.
